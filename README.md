# Velha Game
Jogo da velha no modo console do Windows. 
> Para jogar basta navegar até: `6/bin/debug/6.exe`

> O jogador 1 "X" deve digitar `1` ou `2` para selecionar sua cor, para se diferenciar do jogador 2.
